# webexplorer
A web-based explorer for file upload/download.

# Usage
```
./webexplorer --help
Usage of ./webexplorer:
  -cert string
        https cert file (default "server.crt")
  -d string
        Root Directory (default "./")
  -https
        enable https
  -key string
        https key file (default "server.key")
  -p string
        Port Number (default "8080")
```
